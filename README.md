# 🧮 Mini-project-Purchase-Calculator

Калькулятор покупок на Go с расчётом НДС и стоимости доставки.

## 📝 О проекте

Мой первый проект на языке Go. Программа рассчитывает итоговую стоимость покупки нескольких товаров с учётом НДС (20%) и доставки. При сумме от 3000₽ доставка бесплатная.

## 🚀 Запуск

git clone https://github.com/shrinesdev/Mini-project-Purchase-Calculator.git
cd Mini-project-Purchase-Calculator
go run main.go
